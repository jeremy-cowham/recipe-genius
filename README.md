# Recipe Genius

Lightweight application for finding the best recipes for the ingredients you have.

## Quickstart

Install `pipenv`. On Mac:

```bash
    pip3 install pipenv
```

Windows:

```console
    pip install pipenv
    dir
    cd .
```

Python:

```python
    def add(a, b):
        return a
```

After cloning the repository, you'll need to install depencies with `pipenv`. If you don't already have `pipenv`, you can 

```bash
  pipenv install
```
